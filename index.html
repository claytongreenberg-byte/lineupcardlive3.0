📖 Quick User Guide

🎮 Gameday Card

• Team Dropdown: Select teams to load rosters into lineup

• Player Entry: Manually enter player numbers and names

• Positions: Customize position for each player

• At Bat Checkbox: Mark current batter; next player highlights automatically

👥 Roster Command

• Create Teams: Add new team rosters to your library

• Bulk Import: Paste roster data in format: "# Name Position"

• Edit Details: Add phonetic pronunciations and player info

• Save: Teams appear instantly in Gameday dropdown

🎙️ Script Desk

• Auto-Populated: Current and On-Deck batters appear automatically

• Pronunciation: Add phonetic spellings for announcer ease

• Fun Facts: Store player stats and trivia

• Print-Friendly: Export your script for broadcast booth

⚙️ Admin Suite

• Colors: Customize scoreboard and HUD colors

• Branding: Set conference/school name and logo

• Fonts: Scale text for your broadcast environment

• Backup: Export all data or import previous backups

⚡ Broadcast Controls

• 1st Pitch Timer: Set game start time

• Weather Timer: Countdown for weather delays

• Scoreboard: Update runs, innings with +/− buttons

• Status: Mark game as Pre-Game, Live, Post-Game, or Delayed

💡 Pro Tip: Use the backup feature before each season to save all your rosters and settings!
