📘 QUICK USER GUIDE

🎮 Gameday Card

  • Team Dropdown: Select a team to load its roster into the lineup.
  • Player Slots: Choose a player from the dropdown; name & position fill automatically.
  • Positions: The POS dropdown lets you change the position. Selecting DH shows a "DH For" field.
  • DH For Row: The 10th row is permanently labeled DH For – select your DH here.
  • Pitcher Row: The 11th row is for the starting pitcher.
  • At Bat / On Deck: Click ⬆ AB to mark a batter; next batter highlights automatically.

👥 Roster Command

  • Create, edit, and save teams. Rosters appear instantly in the lineup dropdown.

⏱️ Clocks & Scorebug

  • WX Timer: START a 30‑min countdown (turns red ≤5 min). RESET hides it.
  • Scorebug: Adjust runs and innings (1‑9, then EXTRA) with the button controls.
  • Customize the Scoreboard: Go to ⚙️ Admin Suite → Scoreboard Customization. You can change the team‑name colours, score colour, background, inning‑text colour, and the red accent border. Changes apply instantly and are saved automatically.
